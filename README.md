##### Bel-Quotes

## By: Belinda Ntinyari

## Description
This project was generated with Angular CLI version 13.3.1. Its a web app for creating/adding quotes by filling a form with deatails of the author, the publisher and the quote. 

## Functionalities
Upvote/Downvote
Show Details/Hide Details
Delete quote
Generate new quote
